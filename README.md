<h3>1. 기획 의도</h3>
<p>
프로젝트의 주된 목적은 쇼핑몰이라는 주제를 통해 개발 도구 및 플랫폼을 활용하여 기술적 역량을 향상시키는 것입니다. 
특히, 많은 소비를 유발하는 옷 쇼핑몰을 선택하여 트렌디하면서도 사용자 친화적인 디자인을 추구하였습니다. 
프로젝트는 하나의 웹사이트에 필수적인 서비스를 중점으로 구현하여 사용자들이 편리하게 이용할 수 있도록 설계되었습니다.
</p>
